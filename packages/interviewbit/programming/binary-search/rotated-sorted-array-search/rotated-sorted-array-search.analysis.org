| Question                                                                                 | Answer                                                                                                                                                                      |
|------------------------------------------------------------------------------------------+-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Yeu cau la gi ?                                                                          | Cho 1 mang da duoc sap xep va bi dich chuyen boi 1 doan nao do ta chua biet va target. Tim target trong array A                                                             |
| Yeu cau dac biet gi khong ?                                                              | Khong                                                                                                                                                                       |
| Y tuong gi khong ?                                                                       | Neu nhu mang khong bi dich chuyen thi day la bai toan co dien cua binary search. Neu nhu ta biet duoc so pivot (so dich chuyen) thi ta cung co the tinh duoc de dang        |
| Neu nhu biet pivot thi sao ?                                                             | Thi ta se co chia array thanh 2 phan, phan ben trai pivot la phan lon hon. phan ben phai la phan nho hon. Tu do ta binary search target trong 1 trong 2 doan trai hoac phai |
| Bai toan tro ve tim pivot ?                                                              | Tim pivot ta co the su dung linear de tim nhung dan den do phuc tap la O(n)                                                                                                 |
| Ta biet rang array khong co phan tu nao lap, lai co cach nao khac tot hon de kiem pivot? | Ta co index 0 o array cu chuyn sang array moi co index la pivot. Tu do ta se can tim index cua phan tu nho nhat trong array A                                               |
| Tim nhu nao ?                                                                            | Su dung binary search, pivot se chi co the o trong [0..l-1]                                                                                                                 |
| Viet binary search ra?                                                                   | Don gian, ta se co mid = (start + end) / 2                                                                                                                                  |
| Dieu kien tiep theo la gi ?                                                              | Ta can kiem tra xem lieu array[mid] co phai la phan tu nho nhat ? Ta chon array[l - 1] de so sanh                                                                           |
| Neu nhu array[mid] <= array[l - 1] thi sao ?                                             | Thi ta tim tiep pivot trong [start..mid - 1]                                                                                                                                |
| Neu array[mid] > array[l - 1] ?                                                          | mid khong the la index cua phan tu lon nhat, phai tim tiep trong [mid+1...end]                                                                                              |
| Xong chua?                                                                               | Sau khi kiem duoc pivot thi gan nhu da xong roi                                                                                                                             |
| Phan tich y tuong tren ?                                                                 | Tim kiem pivot ta can O(log(n)), sau khi co pivot tim index cuar target ta lai can O(log(n)) tum vay lai la O(log(n))                                                       |
